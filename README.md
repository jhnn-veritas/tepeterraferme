# TP Terraform - CESI

Ce projet contient la configuration Terraform pour déployer une infrastructure sur AWS.

## Structure des fichiers
- `main.tf` : Configuration principale.
- `variable.tf` : Définition des variables.
- `terraformtp.tf` : Ressources spécifiques au TP.

## Utilisation
1. `terraform init`
2. `terraform plan`
3. `terraform apply`
